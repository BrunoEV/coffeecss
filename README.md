# coffeecss

## Adding coffeecss to your website

``` html
<link rel="stylesheet" type="text/css" href="helpers.css" />
```

``` html
<div class="left">
	Text aligned to the left
</div>
```


``` html
<div class="right">
	Text aligned to the right
</div>
```


``` html
<div class="center">
	Text centered
</div>
```


``` html
<div class="justify">
	Justified text
</div>
```
